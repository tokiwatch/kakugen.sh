# Kakugen (Maxims) CLI

ターミナルを起動するたびに、ランダムに格言や名言を表示してインスピレーションを与えるシンプルなコマンドラインアプリケーションです。

## 特徴

- **高速**: 軽量なBashスクリプトで記述されているため、ターミナル起動時のオーバーヘッドをほとんど生じさせません。
- **依存関係ゼロ**: 標準的なUNIXコマンド（`shuf` / `awk`、`grep`など）のみを使用します。
- **カスタマイズ可能**: 好きなシリーズや言語の格言テキストファイルを自作して追加できます。

## インストール

1. リポジトリをクローンするか、ファイルをダウンロードします。
2. スクリプトに実行権限を付与します:

   ```bash
   chmod +x kakugen.sh
   ```

3. (オプション) `$PATH` の通っているディレクトリ（例: `/usr/local/bin/` や `~/bin/`）にスクリプトを移動し、`kakugen` というコマンドだけで使えるようにします。
   ```bash
   sudo cp kakugen.sh /usr/local/bin/kakugen
   ```

4. データファイル（`sample_ja.txt` 等）を `~/.config/kakugen/` に配置します。（スクリプトと同じディレクトリに置いたままでも動作します）
   ```bash
   mkdir -p ~/.config/kakugen
   cp sample_ja.txt ~/.config/kakugen/
   cp sample_en.txt ~/.config/kakugen/
   ```

## 使い方

スクリプトを実行すると、格言が表示されます:

```bash
./kakugen.sh
```

### オプション

| 短縮 | フルオプション | 説明 | デフォルト値 | 実行例 |
|---|---|---|---|---|
| `-n` | `--number` | 表示する格言の個数 | `1` | `-n 3` |
| `-s` | `--series` | 格言のシリーズ名 | `sample` | `-s jobs` |
| `-l` | `--lang` | 表示する言語 | `ja` | `-l en` |
| `-d` | `--dir` | データファイルのディレクトリ | `~/.config/kakugen/` | `-d /path/to/data` |

### ターミナル起動時に実行する設定

ターミナルを開くたびに格言を表示させるには、`~/.zshrc`（または `~/.bashrc`）の末尾に以下のコマンドを追記します。

```bash
# ~/.zshrc の場合
/path/to/kakugen/kakugen.sh
```

パスを通している場合は以下のように書けます:
```bash
kakugen -n 1 -l ja
```

## オリジナルの格言を追加する

格言のデータはプレーンテキストファイルで管理されます。
1. `<シリーズ名>_<言語>.txt` という名前のファイルを作成します（例: `zen_ja.txt`）。
2. 1行につき1つの格言を記述します。
3. 空行や、`#` から始まる行は無視されます。
4. 作成したファイルを `~/.config/kakugen/` に配置します。
