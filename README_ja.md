# Kakugen (Maxims) CLI

ターミナルを起動するたびに、ランダムに格言や名言を表示してインスピレーションを与えるシンプルなコマンドラインアプリケーションです。

## 特徴

- **高速**: 軽量なBashスクリプトで記述されているため、ターミナル起動時のオーバーヘッドをほとんど生じさせません。
- **依存関係ゼロ**: 標準的なUNIXコマンド（`shuf` / `awk`、`grep`など）のみを使用します。
- **カスタマイズ可能**: 好きなシリーズや言語の格言テキストファイルを自作して追加できます。

## インストール

1. リポジトリをクローンするか、ファイルをダウンロードします。
2. スクリプトに実行権限を付与します:

   ```bash
   chmod +x kakugen.sh
   ```

3. (オプション) `$PATH` の通っているディレクトリ（例: `/usr/local/bin/` や `~/bin/`）にスクリプトを移動し、`kakugen` というコマンドだけで使えるようにします。
   ```bash
   sudo cp kakugen.sh /usr/local/bin/kakugen
   ```

4. データファイル（`sample_ja.txt` 等）を `~/.config/kakugen/` に配置します。（スクリプトと同じディレクトリに置いたままでも動作します）
   ```bash
   mkdir -p ~/.config/kakugen
   cp sample_ja.txt ~/.config/kakugen/
   cp sample_en.txt ~/.config/kakugen/
   ```

## 使い方

スクリプトを実行すると、格言が表示されます:

```bash
./kakugen.sh
```

### オプション

| 短縮 | フルオプション | 説明 | デフォルト値 | 実行例 |
|---|---|---|---|---|
| `-n` | `--number` | 表示する格言の個数 | `1` | `-n 3` |
| `-f` | `--file` | 読み込むファイルパス（カンマ区切りで複数指定可能）。 | なし | `-f my_quotes.txt` |
| `-c` | `--config` | 設定ファイルのパス | `~/.kakugenrc` | `-c ~/.custom_rc` |
| `-s` | `--search` | 指定した文字列を含む格言のみを抽出 | なし | `-s "ことわざ"` |

### 対象ファイルと色の設定 (`~/.kakugenrc`)

デフォルトでは、ホームディレクトリにある `~/.kakugenrc` ファイルに記載されたテキストファイルをすべて読み込みます。
このファイルを作成し、読み込みたいテキストファイルのパスを記述してください。

> **Note**: 複数のテキストファイルを読み込むように設定（または `-f` オプションで複数指定）した場合、どの格言集からの言葉か判別できるように、格言の末尾に自動で出典元（例: `-- sample_ja`）が表示されます。

設定ファイルで `ファイルパス=タイトル` のように `=` で区切って記述することで、ファイル名の代わりにお好みのタイトルを出典として表示させることができます。

また、文字の色をカスタマイズすることも可能です。以下の設定キーを使用します：
- `COLOR_TEXT`: 格言本文の色（デフォルト: `cyan`）
- `COLOR_SYMBOL`: スマートクォート（“ ”）の色（デフォルト: `cyan`）
- `COLOR_TITLE`: タイトル・出典元の色（デフォルト: `gray`）

使用可能な色: `black`, `red`, `green`, `yellow`, `blue`, `magenta`, `cyan`, `white`, `gray` (`none` で色付けなし)

**`~/.kakugenrc` の例:**
```text
# 色のカスタマイズ（省略可能）
COLOR_TEXT=green
COLOR_SYMBOL=yellow
COLOR_TITLE=gray

# 行頭に # をつけるとコメントになります
~/.config/kakugen/sample_ja.txt=日本の格言
~/.config/kakugen/sample_en.txt=English Quotes
```

### ターミナル起動時に実行する設定

ターミナルを開くたびに格言を表示させるには、`~/.zshrc`（または `~/.bashrc`）の末尾に以下のコマンドを追記します。

```bash
# ~/.zshrc の場合
/path/to/kakugen/kakugen.sh
```

パスを通している場合は以下のように書けます:
```bash
kakugen -n 1
```

> **表示に関する注意**: 本アプリケーションは、引用句を美しく見せるためにANSIエスケープシーケンスを用いてイタリック体（斜体）での表示を試みます。お使いのターミナルエミュレーターや選択しているフォントがイタリック体をサポートしていない場合、通常のテキストとして表示されるか、ハイライトされて表示されることがあります。

## オリジナルの格言を追加する

格言のデータはプレーンテキストファイルで管理され、**`%` のみの行を区切り** として扱います。これにより、複数行の格言も正しく表示されます。

1. テキストファイルを作成します。
2. 格言と格言の間を `%` だけで構成された行で区切ります。
3. `~/.kakugenrc` にそのファイルのパスを追記します。

**テキストファイルの記述例:**
```text
ローマは一日にして成らず
%
石の上にも三年
（日本のことわざ）
%
千里の道も一歩から
老子・第64章より
%
```
